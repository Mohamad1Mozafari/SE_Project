# پروژه مهندسی نرم‌افزار ۴۰۴-۴۰۵ دانشگاه کردستان: سامانه پارکینگ عمومی

پروژه زیر در نیمسال دوم درس مهندسی نرم‌افزار ۴۰۴-۴۰۵ در [دانشگاه کردستان](https://uok.ac.ir/)، گروه ۱، با تدریس [دکتر صادق سلیمانی](https://research.uok.ac.ir/~ssulaimany/) به دانشجویان واگذار شده است.

![بنر برنامه](images/README/app_banner.png)

این پروژه بر همکاری تیمی، متدولوژی چابک (AGILE) و مفاهیم کلیدی مهندسی نرم‌افزار تمرکز دارد.

To read the English version of the introduction file, click the link below.

[README انگلیسی](/README.md)

## شروع کار

این دستورالعمل‌ها یک نسخه از پروژه را برای اجرا و آزمایش روی سیستم شما فراهم می‌کنند.

اگر صرفاً قصد اجرای برنامه را دارید، یک نسخه‌ی بسته‌بندی‌شده (packaged) ویندوزی نیز همراه پروژه ارائه شده است. می‌توانید بسته‌ی exe نسخه‌ی x64 را از [اینجا]() دانلود کنید.

### پیش‌نیازها

این پروژه با استفاده از چندین فناوری وب نوشته شده و برای بسته‌بندی برنامه‌ی وب به یک برنامه‌ی ویندوزی محلی، از Electron استفاده می‌کند. برای بهترین سازگاری، مطمئن شوید که نسخه‌ی ابزارهای شما با نسخه‌هایی که پروژه با آن‌ها توسعه یافته مطابقت دارد.

پیش‌نیازهای مورد نیاز برای اجرای پروژه عبارتند از:

- [Node.js](https://nodejs.org/en)

وابستگی‌های (Dependencies) پروژه عبارتند از:

- [React](https://react.dev/)
- [Electron](https://www.electronjs.org/)

### کلون کردن پروژه

مرحله‌ی بعدی، ذخیره‌ی فایل‌های پروژه روی دستگاه شماست. اگر `git` روی دستگاه شما نصب است، می‌توانید به‌راحتی مخزن را کپی کنید:

    git clone https://hamgit.ir/software-engineering/parking-management-system.git --depth 1

**نکته: این دستور فقط آخرین نسخه‌ی هر فایل را روی دستگاه شما کپی می‌کند. اگر به کل تاریخچه‌ی کامیت‌ها نیاز دارید، فلگ `--depth 1` را حذف کنید.**

در غیر این صورت، می‌توانید کد منبع را با [کلیک اینجا](https://hamgit.ir/software-engineering/parking-management-system/-/archive/main/parking-management-system-main.zip?ref_type=heads) به‌صورت فایل فشرده (zip) دانلود کنید.

به پوشه‌ای که فایل‌ها را در خود جای داده بروید:

    cd parking-management-system

سپس باید پیش‌نیازهای اضافی را نصب کنید.

### نصب Node.js

دستورالعمل‌های نصب Node.js بسته به سیستم‌عامل و معماری سیستم شما متفاوت است. برای مشاهده‌ی راهنمای متناسب با نیاز خود، به راهنمای رسمی Node.js مراجعه کنید.

[دانلود Node.js](https://nodejs.org/en/download)

### نصب وابستگی‌ها

پس از راه‌اندازی Node.js روی سیستم خود، مرحله‌ی بعدی نصب وابستگی‌های پروژه است. برای این کار مطمئن شوید که در ریشه‌ی (root) پروژه قرار دارید. سپس دستور زیر را اجرا کنید:

  ```
  npm install
  ```

یا:

  ```
  npm ci
  ```

تا وابستگی‌ها نصب شوند. این فرآیند ممکن است مدتی طول بکشد. پس از آن، اکنون می‌توانید پروژه را اجرا کنید.

## اجرای پروژه

برای اینکه بتوانید پروژه را اجرا کنید، مطمئن شوید که فایل `server.js` که مسئول اتصال برنامه به پایگاه داده است در حال اجراست. برای این کار، در یک ترمینال جداگانه دستور زیر را اجرا کنید:

  ```bash
  node src/app/js_files/server.js
  ```

برای راه‌اندازی برنامه، مطمئن شوید که در پوشه‌ی ریشه‌ی پروژه قرار دارید، سپس دستور زیر را اجرا کنید:

  ```
  npm start
  ```

باید خروجی‌ای مشابه زیر مشاهده کنید:

  ```
  OTHER_DIRECTORY/parking-management-system$ npm start

    > parking-management-system@1.0.0 start
    > vite

    The CJS build of Vite's Node API is deprecated. See https://vite.dev/guide/troubleshooting.html#vite-cjs-node-api-deprecated for more details.

    VITE v5.4.21  ready in 383 ms

    ➜  Local:   http://localhost:5173/
    ➜  Network: use --host to expose
    ➜  press h + enter to show help

  ```

اکنون می‌توانید از طریق مرورگر وب خود، با استفاده از آدرسی که در خروجی نمایش داده شده، به پروژه دسترسی پیدا کنید. در این مورد: ```http://localhost:5173/```

![نمونه‌ای از اجرای پروژه در مرورگر وب](images/README/example_running_webbrowser.png)

اگر ترجیح می‌دهید پروژه را در حالت برنامه (application mode) اجرا کنید، در حالی که **سرور محلی (localhost) در حال اجراست، در یک نشست ترمینال دیگر،** دستور زیر را اجرا کنید:

  ```
  npm run electron
  ```

این کار پروژه را به‌صورت یک برنامه‌ی محلی نمایش می‌دهد.
